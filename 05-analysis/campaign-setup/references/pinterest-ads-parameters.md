# Pinterest Ads — Campaign Parameters

## Context

Pinterest is a visual discovery platform — users actively search for inspiration for future projects.
High purchase intent: 80% of weekly Pinterest users discovered a new product or brand on the platform.
FR audience: predominantly female (60–70%), 25–54 years old, interests: home decor, fashion, cooking, beauty, weddings, gardening, DIY.

**When to use Pinterest Ads:** Visual products with a long purchase cycle (home, fashion, beauty, premium food, weddings, lifestyle training).

## Ad Group Level

### Audience Types

#### Actalike (Pinterest Lookalike)
- Based on site visitors, CRM list, or Pinterest engaged users
- Size: 1× (similar) → 2× (broader)

#### Custom Audiences
| Source | Description |
|---|---|
| Site pixel | Visitors, events |
| Customer list | Uploaded email |
| Pinterest Engagement | People who interacted with your pins |

#### Interest Targeting
Broad categories: Arts & Leisure / Beauty / Design & Home Decor / Fashion / Food / Family / Finance / Gardening / Health / Weddings / Travel / Tech

#### Keyword Targeting
Pinterest = visual search engine. Keyword targeting works like Google Search.
→ Very effective: people type what they want to find.
→ Use broad + specific keywords in combination.

**Match types:**
- Broad: includes variants and synonyms
- Phrase: includes pins containing the phrase
- Exact: only the exact search

#### Demographic Targeting
| Parameter | Options |
|---|---|
| Age | 18–24 / 25–34 / 35–44 / 45–54 / 55–64 / 65+ |
| Gender | All / Female / Male / Non-binary |
| Location | Country, region, city |
| Language | Device language |
| Device | Mobile / Tablet / Desktop |

### Placements

| Placement | Description |
|---|---|
| **Browse (Home / Feed)** | In the Home feed — passive discovery |
| **Search** | In search results — active intent |
| **Related Pins** | Similar pins to what the user is viewing |

**Recommendation:** Activate all 3 at launch. Separate Search and Browse at scale to analyze performance differently.

## Ad Level

### Formats and Specs

| Format | Ratio | Min resolution | Notes |
|---|---|---|---|
| **Standard Pin (Image)** | 2:3 (recommended) / 1:1 | 1000×1500 px | Most common format |
| **Square** | 1:1 | 1000×1000 px | Less visibility than 2:3 |
| **Standard Video** | 2:3 or 9:16 | 1080×1920 | 4s–15min (recommended 6–15s) |
| **Max Video** | 16:9 or 9:16 | 1920×1080 | Full screen — premium |
| **Carousel** | 1:1 or 2:3 | 1000×1000 min | 2–5 cards |
| **Collections** | 1:1 hero + products | 1000×1000 | E-commerce |
| **Idea Pin (Story)** | 9:16 | 1080×1920 | Multi-page — content |

### Text

| Field | Limit |
|---|---|
| Title | 100 characters (30 displayed in preview) |
| Description | 500 characters (50–60 displayed in feed) |

**Pinterest tip:** The title is critical — it's what appears in search results. Naturally integrate target keywords.

## Pinterest Creative Specifics

### What Performs
- **2:3 vertical ratio**: takes up more space in the feed — better visibility
- **Light text overlay**: clear, readable title, cleanly integrated in the image
- **Refined aesthetic**: Pinterest = aspiration. Visual quality > raw authenticity
- **Instructional content**: "How to", "Ideas for", "Guide" — high added value
- **Consistent palette**: pins that adapt to the Pinterest aesthetic perform better
- **Subtle logo**: at the bottom of the image, small — don't interfere with the visual

### What Fails
- Poor quality or pixelated images
- Too much text on the image
- Non-optimized ratio (horizontal = poor visibility)
- Too promotional / "ad" content — Pinterest favors inspiration
- Too aggressive CTA in the foreground

### Pinterest Content Cycle
Pinterest has a long-term effect: a pin can continue generating traffic months after publication.
→ Create evergreen content (no dates, no flash sales)
→ Organically viral pins can be boosted with ad spend

## Pinterest Advertising Policies (Key Points)

- **Alcohol:** Restrictions by country — opt-in required.
- **Adult content:** Prohibited.
- **Beauty products:** Before/after claims accepted if realistic.
- **Finance:** Restrictions on crypto, trading, loans.
- **Copyright:** Do not use images without rights — Pinterest verifies.

**Official resource:** policy.pinterest.com/en/advertising-guidelines
