# Styles Library — Quick Reference (S101 → S604)

Lookup table for all creative style codes. Each code maps to a specific visual execution style with defined format, movement type, and production complexity.

Full style details, Canva templates, and visual examples are in the **Notion Styles Library**. This file is a quick reference for briefs and creative planning.

---

## How Style Codes Work

**Format:** S[Category][Number]
- First digit = Category (1 = Headline, 2 = Collage, 3 = Lifestyle, 4 = Typography, 5 = Data, 6 = Conceptual)
- Second + third digits = Variant within that category

**NC Level** = Nano Banana Complexity (how many prompt iterations needed)
- NC1 = Single prompt, straightforward
- NC2 = 2-3 iterations, moderate complexity
- NC3 = Complex composition, multiple elements

## Carousel Styles

Carousels use a combination of the above styles across slides. Common patterns:

| Pattern | Slide 1 | Slides 2-N | Final Slide |
|---|---|---|---|
| Educational | S401 (Bold Statement) | S103 (Typography) per point | S101 (Product + CTA) |
| Testimonial | S402 (Quote) | S501 (Screenshots) | S204 (Compilation) |
| Before/After | S202 (Split) | S304 (In-Context) | S101 (Product + offer) |
| Listicle | S403 (Data) | S103 per item | S401 (Summary + CTA) |
