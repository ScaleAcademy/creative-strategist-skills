# Creative Strategist Skills

> A systematic skill library for Creative Strategists building high-performance paid social campaigns.

Built by [Scale Academy](https://scale-academy.fr) — the francophone training platform for Creative Strategists.

---

## What This Is

A repository of executable skills and reference documents covering the full Creative Strategist workflow: **Audit → Research → Strategy → Production → Analysis**

Each skill has one job, produces one output, and references what it needs without duplicating it.

## Structure

```
01-audit/       Brand audit — identity, health, alignment
02-research/    Market & audience research — personas, verbatims, reality mapping
03-strategy/    Creative strategy — brief, sequencing, hooks
04-production/  Ad production — static, video, AI video, campaign setup
05-analysis/    Performance analysis — creative, account, strategic gaps
```

## How Skills Work

- **Skill** = executable action that produces a defined output
- **Reference** = knowledge loaded by the skill, never triggered alone
- Skills reference each other via pointers — no duplication
- Status: `wip` | `beta` | `stable`
