# Pinterest Ads — Paramètres de Campagne

## Contexte

Pinterest est une plateforme de découverte visuelle — les utilisateurs y cherchent activement de l'inspiration pour des projets futurs.
Intent d'achat élevé : 80% des utilisateurs hebdomadaires Pinterest ont découvert un nouveau produit ou une nouvelle marque sur la plateforme.
Audience FR : majoritairement féminine (60–70%), 25–54 ans, centres d'intérêt : décoration, mode, cuisine, beauté, mariage, jardinage, DIY.

**Quand utiliser Pinterest Ads :** Produits visuels avec cycle d'achat long (maison, mode, beauté, alimentation premium, mariage, formation lifestyle).

## Niveau Ad Group

### Types d'Audiences

#### Actalike (Lookalike Pinterest)
- Basé sur visiteurs site, liste CRM, ou engagés Pinterest
- Taille : 1× (similaire) → 2× (plus large)

#### Custom Audiences
| Source | Description |
|---|---|
| Pixel site | Visiteurs, événements |
| Liste clients | Email uploadé |
| Engagement Pinterest | Personnes ayant interagi avec tes épingles |

#### Ciblage par Intérêts
Catégories larges : Arts & Loisirs / Beauté / Design & Déco / Mode / Alimentation / Famille / Finance / Jardinage / Santé / Mariage / Voyage / Tech

#### Ciblage par Mots-Clés
Pinterest = moteur de recherche visuel. Le ciblage par mots-clés fonctionne comme Google Search.
→ Très efficace : les gens tapent ce qu'ils veulent trouver.
→ Utiliser des mots-clés larges + spécifiques en combinaison.

**Types de correspondance :**
- Large : inclut variantes et synonymes
- Expression : inclut les épingles contenant la phrase
- Exact : uniquement la recherche exacte

#### Ciblage Démographique
| Paramètre | Options |
|---|---|
| Âge | 18–24 / 25–34 / 35–44 / 45–54 / 55–64 / 65+ |
| Genre | Tous / Femme / Homme / Non-binaire |
| Localisation | Pays, région, ville |
| Langue | Langue du device |
| Appareil | Mobile / Tablette / Desktop |

## Niveau Ad

### Formats et Specs

| Format | Ratio | Résolution min | Notes |
|---|---|---|---|
| **Standard Pin (Image)** | 2:3 (recommandé) / 1:1 | 1000×1500 px | Format le plus courant |
| **Carré** | 1:1 | 1000×1000 px | Moins de visibilité que 2:3 |
| **Vidéo Standard** | 2:3 ou 9:16 | 1080×1920 | 4s–15min (recommandé 6–15s) |
| **Vidéo Maximale** | 16:9 ou 9:16 | 1920×1080 | Plein écran — premium |
| **Carousel** | 1:1 ou 2:3 | 1000×1000 min | 2–5 cartes |
| **Collections** | 1:1 hero + produits | 1000×1000 | E-commerce |
| **Idea Pin (Story)** | 9:16 | 1080×1920 | Multipage — contenu |

### Texte

| Champ | Limite |
|---|---|
| Titre | 100 caractères (30 affichés en preview) |
| Description | 500 caractères (50–60 affichés en feed) |

**Conseil Pinterest :** Le titre est critique — c'est lui qui apparaît dans les résultats de recherche. Intégrer les mots-clés cibles naturellement.

## Spécificités Créatives Pinterest

### Ce qui Performe
- **Ratio 2:3 vertical** : occupe plus d'espace dans le feed — meilleure visibilité
- **Texte overlay léger** : titre clair, lisible, intégré dans l'image ou en surimpression épurée
- **Esthétique soignée** : Pinterest = aspiration. Qualité visuelle > authenticité brute
- **Contenu instructif** : "Comment faire", "Idées pour", "Guide" — forte valeur ajoutée
- **Palette cohérente** : les épingles qui s'adaptent à l'esthétique Pinterest performent mieux
- **Logo discret** : en bas de l'image, petit — ne pas parasiter le visuel

### Ce qui Échoue
- Images de mauvaise qualité ou pixélisées
- Texte trop dense sur l'image
- Ratio non optimisé (horizontal = mauvaise visibilité)
- Contenu trop promotionnel / "pub" — Pinterest favorise l'inspiration
- CTA trop agressif en premier plan

### Cycle de Contenu Pinterest
Pinterest a un effet long terme : une épingle peut continuer à générer du trafic des mois après sa publication.
→ Créer du contenu evergreen (pas de dates, pas d'offres flash)
→ Les épingles organiques virales peuvent être boostées avec de l'ad spend

## Règles Publicitaires Pinterest (Points Clés)

- **Alcool :** Restrictions selon pays — opt-in requis.
- **Contenu adulte :** Interdit.
- **Produits de beauté :** Claims before/after acceptés si réalistes.
- **Finance :** Restrictions sur crypto, trading, prêts.
- **Droits d'auteur :** Ne pas utiliser d'images sans droits — Pinterest vérifie.

**Ressource officielle :** policy.pinterest.com/fr/advertising-guidelines
